Idea de Inmo. Coder

Este sitio web permite la carga y busqueda de casas, departamentos, cocheras y clientes utilizando base de datos. Mediante los botones en la parte superior se puede concretar dichas acciones.

Hecho en Python

Esta es la entrega del proyecto final del curso de Python, dictado por Coderhouse. La idea es aplicar todo lo visto para generar este sitio web, incluyendo funciones, clases, condicionales, bucles, diccionarios, etc. Para mas info: https://www.coderhouse.com/online/python

Integra Django

La totalidad de proyecto se basa en este web framework, en donde los sitios son regenerados tomando las bases de un html "padre" y la visualización de informacion se logra por código "jinja" (encargado de vincular Python con Html.
